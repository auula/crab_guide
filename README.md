## 目的是什么❓
通过连载文章的形式帮助有一定其他语言编程基础的人快速学习和入门 `Rust` 内容包括 `Rust` 基础、内置库、`web` 开发、并发编程等，均来自笔者日常学习和开发经验总结，方便读者理解。

本电子书主要面向有一定其他编程语言开发经验的开发者快速上手 `Rust`，不会涉及到一些非常具体和细节的问题，非零基础教程，编程新手可以先补一补开发基础。
